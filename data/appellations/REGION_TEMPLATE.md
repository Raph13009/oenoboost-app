# NOM_DE_LA_REGION

Utilise ce fichier pour lister les AOP d'une region et les communes associees.

Regles:
- Remplacer `NOM_DE_LA_REGION` par le nom de la region.
- Utiliser un titre `##` par AOP.
- Mettre une commune par ligne avec `- `.
- Utiliser les noms officiels exacts des communes si possible.
- Ne pas ajouter de commentaires inline dans les listes.
- Si une AOP n'a aucune commune pour l'instant, laisser le titre et ajouter `- TODO`.

Exemple:

## Nom AOP 1
- Commune 1
- Commune 2
- Commune 3

## Nom AOP 2
- Commune 4
- Commune 5

## Nom AOP 3
- TODO

Template a remplir:

## NOM_AOP_1
- COMMUNE_1
- COMMUNE_2

## NOM_AOP_2
- COMMUNE_3
- COMMUNE_4

## NOM_AOP_3
- COMMUNE_5
